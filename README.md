<h1><u>Box Score Generator</u></h1>

<p>A tool that utilizes a basketball-reference web scraper, loads all players' statistics from the previous day's games, and sends them via RabbitMQ for later use.</p?

<h3>Installation instructions</h3>
<ul>
    <li>`pip install basketball-reference-web-scraper`</li>
    <li>`pip install pika`</li>
</ul>

<h3>Tools used</h3>
<ul>
    <li>RabbitMQ</li>
    <li>Python3</li>
</ul>
